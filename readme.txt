latest version of python and pip should be installed

framework = python django
database = sqlite

------------------------------------------------------------------------------------

Unzip the folder, opem cmd/terminal in the project directory. 
Install requirements.txt file now
Command : pip install -r requirements.txt
after installing, now run the app
make sure you're in the project directory and you can see manage.py file, now open terminal and write : python manage.py runserver 

website link : localhost:8000
Admin panel link : localhost:8000/admin
admin access : user = admin, password = admin
